# Trouve ton Artisan

Annuaire en ligne permettant la mise en relation entre particuliers et artisans du Bâtiment, Services, Fabrication et Alimentation.

## Fonctionnalités
* **Recherche** : Filtrage des artisans par nom, spécialité et ville.
* **Catégories** : Navigation par secteur d'activité.
* **Fiches Artisans** : Affichage détaillé (Note, Email, Site web, À propos).
* **Gestion d'erreurs** : Page 404


##  Installation local :
1.  Cloner le dépôt.
2.  Installer les dépendances : `npm install`
3.  Lancer le projet : `npm start`
